# SmartGrama Research Project

## Overview
SmartGrama is an AI-driven and blockchain-enabled platform designed for rural microfinance and welfare management in Sri Lanka.

## Main Features
- AI-based loan risk prediction
- Personalized loan recommendation
- Welfare eligibility verification
- Blockchain verification
- Multilingual communication support

## Technologies Used
- Python
- React
- FastAPI
- MongoDB
- Machine Learning

## Installation

### Backend
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
npm install
npm run dev
```

## Research Components
1. Blockchain Layer
2. Smart Contract Automation
3. AI Prediction Engine
4. NLP Communication System
